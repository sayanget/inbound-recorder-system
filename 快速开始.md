# 快速开始指南

## 🚀 三步完成配置

### 第一步：安装依赖包

打开命令行，运行：
```bash
pip install schedule
```

### 第二步：配置邮箱

双击运行 **`配置邮箱.bat`**，按照提示输入：
- 选择邮箱类型（Gmail、QQ、163等）
- 输入发件邮箱和密码/授权码
- 确认收件人邮箱（默认：sayanget@yahoo.com）
- 设置发送时间（默认：每天08:00）

### 第三步：测试或启动

**测试发送：**
- 双击运行 **`测试邮件发送.bat`**
- 检查邮箱是否收到测试邮件

**启动定时服务：**
- 双击运行 **`启动邮件服务.bat`**
- 程序将在后台运行，每天自动发送

---

## 📧 邮箱配置说明

### Gmail
1. 开启两步验证
2. 生成应用专用密码
3. 使用应用专用密码（16位）

### QQ邮箱
1. 设置 → 账户 → POP3/SMTP
2. 开启SMTP服务
3. 获取授权码（不是QQ密码）

### 163邮箱
1. 设置 → POP3/SMTP/IMAP
2. 开启SMTP服务
3. 获取授权码

## 📊 邮件内容

每天会自动发送包含以下内容的邮件：

- **Excel附件**：详细的入库和分拣记录
- **统计数据**：总件数、车辆数等
- **分类统计**：按车辆类型统计

## ❓ 常见问题

**Q: 邮件发送失败？**
- 检查邮箱密码/授权码是否正确
- 确认已开启SMTP服务
- 检查网络连接

**Q: 如何修改发送时间？**
- 重新运行 `配置邮箱.bat`
- 或直接编辑 `email_config.py`

**Q: 如何停止定时任务？**
- 关闭运行窗口
- 或按 Ctrl+C
